# Advanced Invoice Text Extractor

This project extracts key information from invoices using Google's Gemini AI and PyMuPDF.

## Features
- Extract invoice details automatically
- Detect sender and recipient information
- Extract total amount and tax
- Generate annotated invoice PDF

## Technologies Used
- Python
- Google Gemini AI
- PyMuPDF
- python-dotenv

## Setup
1. Install dependencies:
pip install -r requirements.txt

2. Create .env file:
GEMINI_API_KEY=your_api_key

3. Run program:
python main.py
